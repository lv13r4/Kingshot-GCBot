# KingShot Gift Code Automation Bot

A simple tool to automatically find and redeem gift codes for the game KingShot.

### Windows

### Linux / Raspberry Pi
1. Open a terminal in the folder.
2. Run: `chmod +x RUN_ON_LINUX.sh && ./RUN_ON_LINUX.sh`
